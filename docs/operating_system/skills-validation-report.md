# Skills Validation Report

This document records short, reusable validation results for the repo's skill
layer when meaningful skill-routing or governance changes are tested.

## Usage

When a skill change affects repo-wide routing, governance, or other shared
workflow behavior:

- record the scenarios tested
- record any failure that was found
- record the fix that closed the gap
- keep the summary short and reusable

If no formal validation pass has happened yet, leave this file as the template
rather than inventing fictional results.
